# Weather Diary – Дневник погоды

**Автор:** Неверова Антонина
**Дата:** 04.05.26
**Вариант** Weather Diary

## Описание программы

GUI-приложение для ведения дневника погоды. Позволяет:

- Добавлять записи с датой, температурой, описанием и отметкой об осадках.
- Просматривать все записи в таблице.
- Фильтровать записи по дате и по температуре (выше заданного значения).
- Сохранять записи в JSON-файл и загружать из него.
- Проверять корректность ввода (формат даты, числовое значение температуры, непустое описание).

## Как запустить приложение

### Требования
- Установленный Python 3.6 или новее
- Библиотека `tkinter` (обычно идёт в комплекте с Python)

### Шаги

1. Сохраните файл `weather_diary.py` в отдельную папку.
